### 页面功能

#### 1. 总览页面 (Overview)
- **关键指标卡片**: 总资产、今日收益、累计收益、夏普比率
- **权益曲线图**: 带回撤的交互式图表
- **持仓分布饼图**: 各ETF持仓占比可视化
- **策略表现表格**: 三个策略的收益、回撤、夏普对比
- **最近交易记录**: 带颜色标记的买卖记录

#### 2. 实时监控 (Realtime)
- **ETF价格卡片**: 4只核心ETF的实时价格、涨跌幅
- **实时走势图**: 分时图支持多周期切换
- **买卖盘深度**: 5档买卖挂单实时展示
- **成交明细**: 逐笔成交记录带方向颜色
- **板块热力图**: 各行业板块涨跌热度

#### 3. 回测分析 (Backtest)
- **回测配置面板**: 策略选择、日期范围、初始资金
- **KPI指标卡片**: 总收益、年化收益、夏普、最大回撤、胜率、盈亏比
- **权益曲线与回撤**: 双Y轴组合图表
- **月度收益热力图**: 年度x月度收益矩阵
- **收益分布直方图**: 日收益频率分布
- **交易记录表格**: 完整的成交明细

#### 4. 策略信号 (Signals)
- **L1-L4概览卡片**: 各层得分、状态、颜色标识
- **综合评分仪表盘**: 0-100分圆形仪表盘
- **雷达图**: L1-L4+动量+波动率六维展示
- **当前交易信号**: 触发条件、建议仓位、止损设置
- **L1-L4指标详情表**: 完整指标列表、数值、评分、权重、趋势

#### 5. 风险控制 (Risk)
- **风险概览卡片**: 整体等级、仓位使用、日回撤、未处理告警
- **风险限额进度条**: 6项限额的使用进度和百分比
- **风险指标历史**: 回撤、波动率、VaR的时间序列
- **风险告警列表**: 级别、类型、描述、当前值、阈值、状态

### 技术特性

#### UI设计
- **深色主题**: 专为交易监控优化的深色配色
- **响应式布局**: 基于Bootstrap的网格系统
- **Plotly图表**: 交互式、高性能的金融图表
- **实时刷新**: 5秒间隔自动更新

### 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 启动仪表板
python run_dashboard.py

# 指定端口
python run_dashboard.py --port 8080

# 允许外部访问
python run_dashboard.py --host 0.0.0.0

# 调试模式
python run_dashboard.py --debug
```

访问地址: http://127.0.0.1:8050

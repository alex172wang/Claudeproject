# 多维量化指标体系 v1.0

> **核心理念：** 单一趋势指标只是"一个截面的投影"。
> 本体系从 **趋势、结构、共振、缺口** 四个维度拼合更完整的市场图景，
> 覆盖 ETF轮动 / 永久组合 / 主题仓位 三个策略层。

---

## 一、框架总览

```
┌───────────────────────────────────────────────────────────┐
│                      决策输出层                             │
│    ETF轮动信号  ←  永久组合再平衡  ←  主题仓位择时          │
├───────────────────────────────────────────────────────────┤
│                      信号合成层                             │
│           多维度评分 → 加权综合 → 阈值触发                  │
├──────────────┬──────────────┬─────────────┬───────────────┤
│  L1 趋势层   │  L2 结构层    │  L3 共振层   │  L4 缺口层    │
│  (方向)      │  (品质)       │  (关联)      │  (风险)       │
│              │              │             │               │
│  概率最高    │  投影的       │  多投影间    │  信息损失      │
│  的路径      │  纹理         │  的交汇      │  的度量        │
├──────────────┼──────────────┼─────────────┼───────────────┤
│  mootdx      │  mootdx      │  AKShare    │  期权数据      │
│  AKShare     │  AKShare     │  FRED       │  AKShare      │
│              │              │             │  FRED         │
└──────────────┴──────────────┴─────────────┴───────────────┘
```

**策略适用矩阵：**

| 指标层 | ETF轮动 | 永久组合 | 主题仓位 |
|--------|---------|---------|---------|
| L1 趋势 | ★★★ 核心驱动 | ★☆ 辅助参考 | ★★★ 入场依据 |
| L2 结构 | ★★ 信号过滤 | ★★ 再平衡触发 | ★★ 品质确认 |
| L3 共振 | ★★★ 轮动依据 | ★★★ 配比调整 | ★★ 主题验证 |
| L4 缺口 | ★★ 风控叠加 | ★★★ 防御触发 | ★★★ 止损/对冲 |

## 二、L1 趋势层——"概率最高的路径"

> 对应现有策略核心逻辑，保留并规范化。
> 哲学映射：每个投影沿概率最大的路径运动。

### 指标清单

| 编号 | 指标 | 计算方法 | 默认参数 | 数据源 |
|------|------|---------|---------|--------|
| L1-01 | 复合斜率动量 | 30日斜率×R²(0.6) + 15日斜率×R²(0.4) | 30d/15d, 权重0.6/0.4 | mootdx |
| L1-02 | EMA趋势过滤 | 收盘价 > EMA(N) 为多头环境 | N=120 | mootdx |
| L1-03 | 趋势加速度 | 复合斜率动量的一阶差分（动量的动量） | 5日差分窗口 | 派生 |
| L1-04 | 价格通道位置 | (Close - N日最低) / (N日最高 - N日最低) | N=60 | mootdx |
| L1-05 | FRED趋势共振 | 美10Y国债收益率斜率 + 美元指数DXY斜率 | 30d斜率 | FRED |

### 使用逻辑

- **ETF轮动：** L1-01为排序主键，L1-02为入选门槛，L1-03用于同分排序（加速中的趋势优先）。
- **永久组合：** L1-04标记各资产在自身波动区间的位置，辅助判断再平衡方向。
- **主题仓位：** L1-01 + L1-02双确认入场，L1-05验证宏观环境是否支撑主题逻辑。

## 三、L2 结构层——"投影的纹理"

> 两段同斜率的上涨，一个平滑一个震荡，高维结构完全不同。
> 哲学映射：同一方向的投影，内部纹理揭示本体的不同面。

### 指标清单

| 编号 | 指标 | 计算方法 | 默认参数 | 数据源 |
|------|------|---------|---------|--------|
| L2-01 | Hurst指数 | R/S分析法，衡量趋势持续性品质 | 窗口=60d | mootdx |
| L2-02 | 波动率结构比 | 短期实现波动率 / 长期实现波动率 | 5d / 30d | mootdx |
| L2-03 | 成交量形态分歧 | 价格新高时成交量相对MA(20)的偏离度 | MA(20) | mootdx |
| L2-04 | 回撤分形维度 | 最大回撤序列的Hurst指数（回撤是否有自相似性） | 滚动120d | 派生 |
| L2-05 | K线实体比 | N日内实体占比均值（实体/振幅），衡量趋势"纯净度" | N=20 | mootdx |
| L2-06 | 波动率自相关 | 日收益率绝对值的ACF(1)，衡量波动率聚集程度 | lag=1, 窗口30d | mootdx |

### 解读框架

| 指标 | 高值含义 | 低值含义 |
|------|---------|---------|
| L2-01 Hurst | >0.5 趋势可持续，动量策略友好 | <0.5 均值回复，动量信号易失效 |
| L2-02 波动率结构比 | >1.0 短期波动放大，结构不稳 | <1.0 短期平静，趋势平滑推进 |
| L2-03 量价分歧 | 量能确认，突破可信 | 量能背离，趋势可能衰竭 |
| L2-05 K线实体比 | 趋势纯净，推动力强 | 影线多，犹豫/争夺激烈 |
| L2-06 波动率自相关 | 波动率有惯性，风险管理需加宽 | 波动率随机，可用标准止损 |

### 使用逻辑

- **ETF轮动：** L2-01 > 0.5作为信号过滤器——只在趋势可持续的品种上执行动量策略。L2-02 < 1.2过滤"虚假动量"。
- **永久组合：** L2-04异常（回撤分形维度突变）触发非常规再平衡。
- **主题仓位：** L2-03量价确认 + L2-05实体比 > 0.6双重确认后才加仓。

## 四、L3 共振层——"多投影间的交汇"

> 单资产是单个投影，资产间的关联模式才是高维结构的轮廓。
> 哲学映射：同源投影沿概率最高路径运动，必然交汇。

### 指标清单

| 编号 | 指标 | 计算方法 | 默认参数 | 数据源 |
|------|------|---------|---------|--------|
| L3-01 | 滚动相关性矩阵 | 候选池内所有资产两两滚动Pearson相关系数 | 窗口=60d | AKShare |
| L3-02 | 相关性变速 | L3-01的一阶差分，捕捉相关性瓦解/收敛速度 | 5d差分 | 派生 |
| L3-03 | PCA第一主成分解释率 | 候选池收益率矩阵PCA，PC1方差解释比 | 窗口=60d | 派生 |
| L3-04 | 跨市场动量一致性 | A股/港股/美股/商品同方向动量占比 | 30d斜率符号 | AKShare |
| L3-05 | 宏观-资产共振度 | FRED指标（10Y利率、CPI、PMI）与资产收益的滚动相关 | 窗口=90d | FRED+AKShare |
| L3-06 | 板块轮动速度 | 候选池内排名变化的标准差（排名越稳定，轮动越慢） | 滚动4周 | 派生 |

### 关键信号解读

**L3-03 PCA解释率——"维度探测器"：**

- **PC1 > 70%：** 市场被单一因子驱动（系统性风险/流动性），所有资产高度同步。轮动策略失效，分散化暂时失效，应降仓或对冲。
- **PC1 40%-70%：** 正常区间，资产有差异但存在共同趋势。轮动策略最佳工作区间。
- **PC1 < 40%：** 市场高度分散，每个资产走自己的路。轮动信号可信度高，可以更集中持仓。

**L3-02 相关性变速——"相遇预警"：**

- 相关性快速收敛（多资产同时变得相关）→ 系统性事件正在形成，减仓信号。
- 相关性快速瓦解 → 此前的"交汇"结束，新的分散机会出现。

### 使用逻辑

- **ETF轮动：** L3-06轮动速度过快时降低换手频率。L3-03过高时暂停轮动，切入现金或债券。
- **永久组合：** L3-01 + L3-05共同驱动配比调整。股债相关性转正时增配黄金/商品。L3-05追踪宏观因子切换，调整配比逻辑。
- **主题仓位：** L3-04跨市场一致性确认主题全球性（如能源主题，需原油+能源股+煤炭同向）。

## 五、L4 缺口层——"信息损失的度量"

> 投影必然丢失信息，丢失的部分在市场中体现为隐含风险。
> 哲学映射：三维投影与高维本体之间的裂缝越大，惊喜/惊吓越近。

### 指标清单

| 编号 | 指标 | 计算方法 | 默认参数 | 数据源 |
|------|------|---------|---------|--------|
| L4-01 | 隐含-实现波动率价差 | 50ETF期权IV - 标的30d实现波动率 | 平值期权 | AKShare期权 |
| L4-02 | 期权偏度 | 25Δ看跌IV - 25Δ看涨IV（恐惧的不对称性） | 当月合约 | AKShare期权 |
| L4-03 | 认沽认购比 | 看跌期权成交量 / 看涨期权成交量 | 日度数据 | AKShare期权 |
| L4-04 | 流动性缺口 | 买一卖一价差/中间价的滚动均值 | N=20d | mootdx |
| L4-05 | 尾部风险度量 | 滚动N日收益率的峰度（尾巴有多肥） | N=60d | mootdx |
| L4-06 | 跳空缺口频率 | N日内跳空幅度>1%的天数占比 | N=30d | mootdx |
| L4-07 | FRED压力合成 | 美联储资产负债表变化率 + 信用利差(BAA-AAA)趋势 | 月度/周度 | FRED |

### 风险状态矩阵

| 状态 | L4-01 IV价差 | L4-02 偏度 | L4-05 峰度 | 解读 | 行动 |
|------|-------------|-----------|-----------|------|------|
| 平静 | < 3% | < 5% | < 4 | 市场定价合理 | 正常执行策略 |
| 隐忧 | 3-8% | 5-10% | 4-6 | 有人在买保险 | 收紧止损，减少杠杆 |
| 恐慌酝酿 | > 8% | > 10% | > 6 | 尾部风险定价激增 | 降仓，启动对冲 |
| 恐慌释放 | 高位回落 | 高位回落 | 开始下降 | 风险已释放 | 逆向布局窗口 |

### 使用逻辑

- **ETF轮动：** L4-01 + L4-05构建"风险覆盖层"。缺口指标进入"隐忧"区间，轮动目标自动偏向防御资产。
- **永久组合：** L4-07 FRED压力合成作为"宏观缺口"指标。缩表加速+利差走阔=流动性缺口扩大，触发债券/黄金比例上调。
- **主题仓位：** L4-02偏度是主题仓位的生死线。即使趋势指标看好，期权偏度飙升也应减仓。

## 六、信号合成层——四维拼图

### 6.1 单品种综合评分

每个候选资产获得四个维度的标准化得分（0-100）：

```
综合得分 = L1得分 × W1 + L2得分 × W2 + L3得分 × W3 + L4得分 × W4

默认权重（ETF轮动）:  W1=0.40, W2=0.20, W3=0.20, W4=0.20
默认权重（永久组合）:  W1=0.15, W2=0.20, W3=0.35, W4=0.30
默认权重（主题仓位）:  W1=0.35, W2=0.25, W3=0.15, W4=0.25
```

### 6.2 维度评分细则

**L1得分：**
```
L1 = rank(复合斜率动量) × 0.6 + 价格通道位置 × 0.2 + 趋势加速度标准化 × 0.2
前置条件：必须通过EMA120过滤，否则直接归零
```

**L2得分：**
```
L2 = Hurst标准化 × 0.3 + (1 - 波动率结构比标准化) × 0.25
   + 量价确认 × 0.2 + 实体比 × 0.15 + (1 - 波动率自相关) × 0.1
惩罚：Hurst < 0.45时整体打五折
```

**L3得分：**
```
L3 = (1 - 与大盘相关性) × 0.3 + 跨市场一致性 × 0.3
   + (1 - PC1解释率) × 0.2 + (1 - 轮动速度) × 0.2
注：不同策略中含义不同——轮动看分散性，永久组合看配比合理性
```

**L4得分（反向——缺口越大得分越低）：**
```
L4 = (1 - IV价差标准化) × 0.25 + (1 - 偏度标准化) × 0.25
   + (1 - 峰度标准化) × 0.2 + (1 - 流动性缺口) × 0.15
   + (1 - 跳空频率) × 0.15
```

### 6.3 策略触发规则

**ETF轮动（周度执行）：**
1. 全候选池计算综合得分
2. 选取得分最高且通过L1-02门槛的品种
3. 若L3-03 > 70%（系统性风险），切入货币基金/短债
4. 若L4状态为"恐慌酝酿"，强制加入黄金ETF至50%仓位

**永久组合（月度/事件驱动再平衡）：**
1. 常规月度检查各资产偏离目标配比是否超过阈值
2. L3-01相关性结构变化触发非常规再平衡
3. L4-07 FRED压力信号触发防御性调整
4. L2-04回撤分形异常触发紧急检视

**主题仓位（事件驱动）：**
1. 入场：L1双确认 + L2品质通过 + L3跨市场一致
2. 加仓：综合得分持续上升 + L4状态为"平静"
3. 减仓：L4偏度飙升 OR L3跨市场一致性瓦解
4. 清仓：L1-02失守（跌破EMA120）

## 九、L5 元层——执行偏差日志

> 每次策略偏离不是噪音，而是尚未被量化的高维信号。

### 记录字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 日期时间 | 偏离发生时间 | 2025-03-27 14:30 |
| 计划行动 | 策略信号要求的操作 | 轮动切换至黄金ETF |
| 实际行动 | 实际执行的操作 | 犹豫，只换了50%仓位 |
| 偏离类型 | 提前/延迟/部分/反向 | 部分执行 |
| 主观原因 | 当时的直觉/感受 | 觉得黄金涨太快了 |
| 事后验证 | 7天后回看结果 | 对了/错了/中性 |
| 复盘标注 | 是否反映未量化的合理信号 | 是——黄金短期超买确实回调了 |

**长期价值：** 累计足够样本后统计"直觉胜率"。若某类偏离胜率持续 > 60%，说明直觉在捕捉尚未进入L1-L4的信号维度，值得将其量化纳入系统。

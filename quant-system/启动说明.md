# 量化交易系统 - 启动说明

## 快速启动（推荐）

### Windows 用户

双击以下任一脚本即可：

| 脚本 | 说明 |
|------|------|
| `启动系统.bat` | 完整启动：初始化环境 + 数据同步 + Dashboard |
| `启动Dashboard.bat` | 仅启动 Dashboard（环境已初始化时使用）|

### 命令行启动

```bash
# 进入项目目录
cd quant-system

# 方式1：完整启动（推荐首次使用）
python scripts/start_quant_system.py

# 方式2：仅启动 Dashboard
python run_dashboard.py

# 方式3：仅初始化环境
python scripts/start_quant_system.py --init-only
```

## 启动参数

### scripts/start_quant_system.py

| 参数 | 说明 |
|------|------|
| `--no-browser` | 不自动打开浏览器 |
| `--skip-checks` | 跳过环境检查 |
| `--init-only` | 仅初始化环境，不启动服务 |

### run_dashboard.py

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--host` | 主机地址 | 0.0.0.0 |
| `--port` | 端口号 | 8050 |
| `--debug` | 调试模式 | False |

## 访问地址

启动成功后，可以通过以下地址访问：

| 服务 | 地址 | 说明 |
|------|------|------|
| **Dashboard** | http://localhost:8050 | 实时监控仪表板 |
| Django Admin | http://localhost:8000/admin/ | 后台管理（如需启动） |
| API | http://localhost:8000/api/ | REST API（如需启动） |

### 默认管理员账号

- 用户名: `admin`
- 密码: `admin123`

## 系统特性

### 自动数据同步

- **实时行情**: 每 5 秒自动同步
- **K线数据**: 按需获取并缓存 1 小时
- **ETF列表**: 从 `quant_system/settings.py` 配置读取

### 配置 ETF 池

编辑 `quant-system/quant_system/settings.py` 中的 `QUANT_SYSTEM` 配置：

```python
QUANT_SYSTEM = {
    'etf_pool': [
        {'code': '510300', 'name': '沪深300ETF', 'category': 'broad', 'market': 'sh'},
        {'code': '159915', 'name': '创业板ETF', 'category': 'broad', 'market': 'sz'},
        # 添加更多 ETF...
    ],
    'data_sync': {
        'realtime_interval': 5,    # 实时行情同步间隔（秒）
        'kline_ttl': 3600,         # K线缓存时间（秒）
    }
}
```

## 故障排除

### 端口被占用

如果 8050 端口被占用，可以指定其他端口：

```bash
python run_dashboard.py --port 8051
```

### 依赖安装失败

手动安装依赖：

```bash
pip install -r requirements.txt
```

### 数据获取失败

检查网络连接，确保能访问东方财富等数据源。

## 开发模式

如需启动 Django 开发服务器（用于 API 或 Admin）：

```bash
# 启动 Django 服务器
python manage.py runserver

# 在另一个终端启动 Dashboard
python run_dashboard.py
```
